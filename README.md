# Data-Analytics-Projects

**Project Overview**
This project is an interactive Power BI dashboard that analyzes credit card defaulter behavior using a real-world dataset. The goal is to identify patterns and trends in credit card defaults across key demographics including education level, gender, age group, and marital status, and to provide actionable recommendations for the financial sector.


**Dataset**
**Source:** Kaggle - Default of Credit Card Clients Dataset
**Records:** 30,000 customers
**Features:** Demographics, credit limit, payment history, bill amounts (6 months)
**Used for**: Practice and educational purposes only
