
# HR People Analytics — Employee Attrition Analysis

## Project Overview
Analyzed 1,470 employee records to identify key attrition 
drivers and support workforce retention strategies.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn) — EDA and visualization
- SQL Server (SSMS) — business analysis and stored procedures

## Key Findings
- Overall attrition rate: 16.12%
- Sales department highest attrition: 20.63%
- Overtime employees leave 3x more (30% vs 10%)
- New employees year 1 highest churn: 34.88%
- Leavers earn $2,000/month less than stayers
- Age group 18-25 highest attrition: 34.78%

## SQL Skills Demonstrated
- GROUP BY, ORDER BY, CASE WHEN
- Subqueries and CTEs
- Stored Procedures with parameters
- Indexes for query optimization

## Project Structure
- Python notebook with 12+ visualizations
- 8 SQL analyses plus stored procedure
- Cleaned dataset with 1,470 employee records
